# DISPLAY

## block

Ocupa todo o espaço da linha, empurrando o restante para baixo

---

## inline

Ocupa estritamente apenas o espaço do elemento, não aceitando as propriedades de altura (heigth) e largura (width)

---

## inline-block

Uma mistura do `inline` com o `block`, ocupando apenas o espaço do elemento, mas aceitando largura (width) e altura (heigth)

---

## flex

Possibilita um controle maior sobre os elementos, possibilitando posicionar os elementos com mais facilidade (direita, esquerda, centro, espalhado). Utilizado para layouts unidimensionais.

---

## grid

Divide o espaço em linhas e colunas, sendo utilizado em layouts bidimensionais

---

## none

Some com os elementos, tornando-os invisíveis
